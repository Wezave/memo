# Игра Мемо
Игра для тренировки и развития памяти.

## Описание проекта
С помощью HTML, CSS (препроцессора SAAS), JS создан сайт с игрой. 
Сайт содержит главное меню с возможностью выбора условий игры и саму игру.
Логика реализована на чистом JS, верстка сайта адаптивна (подходит как для больших экранов, так и для смартфонов) и доступна (адекватное изучение сайта скринридерами: используются подходящие, не универсальные, блоки, альтернативы для изображений, структура страницы).  

## Структура проекта
1. `index.html` — сама страница
2. `styles/*.css` и `styles/*.scss` — стили страниц и переменные
3. `fonts/*.css` — шрифты, используемые на сайте
3. `script.js` — вся логика работы игры и переключения страниц
